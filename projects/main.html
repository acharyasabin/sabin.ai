Projects




⭐ Real Time Facial Skin Segmentation using Deep Convolutional Network
-- The project is about the implementation of a custom semantic segmentation model that segments the facial region of a face (excluding eyebrows, eyes, nose, lips, and mouth) in real-time. There are various applications of facial skin segmentation like emotion classification, heart rate detection and many more.
⭐ Contactless Heart Rate Measurement from Facial Videos
-- The project is about the detection of heart rate from facial videos by examining the subtle changes in facial pixels of video frames.
⭐ Semantic Segmentation of Skin Lesions using Light Weight Convolutional Network
-- The project is about the implementation of custom semantic segmentation model that has only 2.318 million of parameters that achieved an average threshold Jaccard index of 83.79% on the validation dataset of “ISIC 208: Skin Lesion Analysis Towards Melanoma Detection” which is comparable but also outperforms certain state-of-the-art segmentation models.
⭐ Real Time Vehicle Tracking and Counting
-- The project is about tracking numerous vehicles on the road and counting the number of vehicles traveling through opposite lanes of a road to measure traffic flow.
